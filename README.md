# 🎤 Zingen

> Projeto desenvolvido como parte do Curso Fullstack da Rocketseat.

O **Zingen** é uma *landing page* moderna com um tema escuro (dark mode), concebida para apresentar uma aplicação de canto e karaoke. Com o lema "Quem canta os seus males espanta", a página foca-se numa interface de utilizador (UI) atrativa, direta e apelativa para amantes de música.

## 🔗 Acesso ao Projeto

O projeto encontra-se alojado e disponível através do GitHub Pages!  
👉 https://thiagomarcelinovieira.github.io/zingen-rocketseat/

---

## ⚙️ Características

- Interface de utilizador moderna e elegante, focada num design escuro.
- Navegação simples com chamadas de atenção (CTAs) claras ("Ver planos", "Baixar agora").
- Secções informativas preparadas para apresentar as funcionalidades da aplicação, planos e preços.

---

## 🛠️ Tecnologias Utilizadas

Com base na estrutura do repositório, este projeto foi construído utilizando as seguintes tecnologias web:

- HTML5: Estruturação semântica e acessível da página.
- CSS3: Estilização, layout e adaptação visual dos elementos (presente na pasta `styles`).

---

## 🚀 Como executar o projeto localmente

Se desejar correr este projeto na sua própria máquina, basta seguir os passos abaixo:

1. Clone o repositório:
```bash
git clone https://github.com/ThiagoMarcelinoVieira/zingen-rocketseat.git
```

2. Aceda à pasta do projeto:
```bash
cd zingen-rocketseat
```

3. Abra no seu navegador:

Abra o ficheiro `index.html` com duplo clique ou utilize uma extensão como o Live Server no VS Code.
